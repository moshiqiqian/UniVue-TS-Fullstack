## 项目介绍

本项目为前后端分离项目提供基础框架。

特点：

1. 多平台开发：使用uni-app，可以实现多个平台，一份代码。
2. 支持通过网络代理，来访问API：使用SOCKS 代理访问外部网络，可以接入需要代理才能使用的API。

### 前端框架

本框架的前端基于 **uni-app** 和 **Vue 3** 。

#### 主要依赖

| **特点**      | **详细说明**                                                 | **依赖（package.json）**                              |
| ------------- | ------------------------------------------------------------ | ----------------------------------------------------- |

| **跨端框架**  | **uni-app：** 它是 DCloud 提供的核心跨端框架，负责将您的 Vue 代码编译成 App、H5、小程序等平台可识别的代码。 | `@dcloudio/uni-app` 等一系列 `@dcloudio/uni-*` 依赖。 |

| **状态管理**  | **Pinia：** 这是一个官方推荐的 Vue 状态管理库，它更简洁、更类型友好，用于管理应用中的共享状态数据（例如用户登录信息）。 | `"pinia": "^2.3.1"`。                                 |
| **UI 组件库** | **uni-ui：** uni-app 官方提供的 UI 组件库，可以直接使用封装好的组件快速构建界面。 | `"@dcloudio/uni-ui": "^1.5.11"`。                     |
| **开发语言**  | **TypeScript：** 基础配置中包含了 TypeScript，用于提供静态类型检查，增强代码健壮性和可维护性。 | `"typescript": "^4.9.4"`, `"vue-tsc"`。               |

### 后端框架

#### 主要依赖

本框架的后端基于 **Express + TypeScript** 。

| **特点 (Feature)** | **详细说明 (Detailed Explanation)**                          | **依赖 (Package)**                                           |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Web 框架**       | 使用最流行的 **Express.js** 框架来构建 RESTful API 和处理 HTTP 请求，具有轻量、灵活的特点。 | `express@^5.1.0`                                             |
| **开发语言**       | 使用 **TypeScript** 语言进行开发，提供静态类型检查，增强代码健壮性和可维护性。 | `typescript@^5.9.3`                                          |
| **运行时支持**     | 通过 **ts-node**，允许 Node.js 直接运行 `.ts` 源代码，无需手动编译，极大地提高了开发调试效率。 | `ts-node@^10.9.2`                                            |
| **数据库连接**     | 使用高效的 **`mysql2`** 驱动程序，用于连接和操作 MySQL 或 MariaDB 数据库，是项目的数据存储层核心。 | `mysql2@^3.15.3`                                             |
| **跨域通信**       | 配置了 **CORS** 中间件，解决了前端 uni-app 在不同端口或域名下访问后端时遇到的跨域请求限制。 | `cors@^2.8.5`                                                |
| **HTTP 客户端**    | 依赖 **Axios**，表示后端程序具有向外部第三方 API 或其他微服务发送 HTTP 请求的能力。 | `axios@^1.13.2`                                              |
| **唯一标识符**     | 依赖 **UUID** 库，常用于生成通用唯一标识符，确保数据库记录或临时文件拥有不重复的 ID。 | `uuid@^13.0.0`                                               |
| **代理支持**       | 具有通过 **SOCKS 代理**访问外部网络的能力，适用于特定网络环境或需要代理请求的场景。 | `socks-proxy-agent@^8.0.5`                                   |
| **类型声明 (Dev)** | 为 Express、Node.js、CORS 和 UUID 等核心库提供了 **TypeScript 类型定义**，确保在使用第三方库时有准确的类型提示和检查。 | `@types/express@^5.0.5`, `@types/node@^24.10.0`, `@types/cors@^2.8.19`, `@types/uuid@^10.0.0` |
